# brand.md — แนวทางแบรนด์ของ BUMBIMBANANA Fashion Product Radar

เอกสารนี้กำหนดแนวทางแบรนด์ (Brand Guideline) สำหรับเว็บไซต์ **BUMBIMBANANA Fashion Product Radar** เพื่อให้การออกแบบหน้าจอทุกหน้ามีความสอดคล้องกัน ทั้งโทนสี ตัวอักษร ระยะห่าง และน้ำเสียงการสื่อสาร

---

## 1. ชื่อแบรนด์และความหมาย

* ชื่อเต็ม: **BUMBIMBANANA Fashion Product Radar**
* "Fashion Product Radar" หมายถึง เรดาร์ (Radar — ระบบตรวจจับ/ค้นหาสัญญาณ) สำหรับตรวจจับสินค้าแฟชั่นที่น่าสนใจจาก TikTok Shop (แพลตฟอร์มขายสินค้าออนไลน์บน TikTok)
* ใช้เป็นชื่อเรียกในหัวเว็บไซต์ (Header — ส่วนหัวของหน้าเว็บ), หน้าแท็บเบราว์เซอร์ (Browser Tab) และเอกสารทุกฉบับ

## 2. บุคลิกแบรนด์ (Brand Personality)

| ลักษณะ | คำอธิบาย |
|---|---|
| แฟชั่นพรีเมียม | ใช้พื้นที่หายใจเยอะ ไม่อัดข้อมูลแน่นเกินไป |
| เรียบหรู | สีพื้นฐานเรียบ ไม่ฉูดฉาด ใช้สีเน้น (Accent Color) อย่างประหยัด |
| ดูเป็นมืออาชีพ | ตัวเลขสำคัญ เช่น ยอดขาย รายได้ ค่าคอมมิชชัน ต้องอ่านง่ายและเชื่อถือได้ |
| อ่านข้อมูลง่าย | จัดลำดับข้อมูลชัดเจน ใช้ตารางบนคอมพิวเตอร์ และการ์ดบนมือถือ |
| มีความเป็นแฟชั่นเกาหลี | เส้นสายเรียบง่าย โทนสีนวล ขอบมน ไม่ใช้สีตัดกันแรง |
| ไม่ดูเป็นระบบบัญชีที่แข็งเกินไป | หลีกเลี่ยงตารางทึบหนา สีเทาเข้มทั้งหน้า หรือฟอนต์ราชการ |

## 3. โทนสี (Color Palette)

| บทบาท | สี | การใช้งาน |
|---|---|---|
| พื้นหลังหลัก (Background) | ขาว `#FFFFFF` | พื้นหลังหน้าเว็บหลัก |
| พื้นหลังรอง (Secondary Background) | ครีมอ่อน `#F8F4EC` | ส่วนแบ่งโซน, การ์ดพื้นหลังอ่อน |
| ข้อความหลัก (Primary Text) | ดำเกือบสนิท `#1A1A1A` | หัวข้อ, ตัวเลขสำคัญ |
| ข้อความรอง (Secondary Text) | เทา `#6B6B6B` | คำอธิบาย, label ย่อย |
| เส้นขอบ/แบ่งโซน (Border) | เทาอ่อน `#E5E1D8` | ขอบการ์ด, เส้นแบ่งตาราง |
| สีเน้น (Accent Color) | ทองอ่อน/เบจพรีเมียม `#C9A877` | ปุ่มหลัก, ตัวเลขเด่น, ไอคอนสำคัญ, แถบไฮไลต์ |
| สีเน้นเข้ม (Accent Hover) | ทองเข้มขึ้น `#B6924F` | สถานะ hover/active ของปุ่ม |
| สถานะเติบโต (Positive/Growth) | เขียวมรกตนวล `#3E7A5B` | เปอร์เซ็นต์เติบโตที่เป็นบวก |
| สถานะลดลง (Negative) | แดงอิฐนวล `#B5564B` | เปอร์เซ็นต์เติบโตที่เป็นลบ |

หลักการใช้สี: ห้ามใช้สีเน้น (ทอง) มากกว่า 1 องค์ประกอบหลักต่อหน้าจอ เพื่อให้สีเน้นยังคงความรู้สึก "พรีเมียม" ไม่ใช่สีพื้นทั่วไป

## 4. ฟอนต์ (Typography)

* ใช้ฟอนต์ภาษาไทยที่อ่านง่ายและดูทันสมัย เช่น **Noto Sans Thai** หรือ **IBM Plex Sans Thai** (โหลดผ่าน `next/font`)
* หัวข้อ (Heading): น้ำหนักตัวอักษรหนา (Bold/Semibold), ขนาดใหญ่ชัดเจน
* ตัวเลขสำคัญ (ยอดขาย, รายได้, ค่าคอมมิชชัน, % เติบโต): ใช้ขนาดใหญ่กว่าข้อความรอบข้างเสมอ น้ำหนักหนา และใช้สีเน้นหรือสีดำเข้ม เพื่อให้ "ตัวเลขเด่น" ตามที่กำหนด
* ข้อความรอง/คำอธิบายภาษาอังกฤษในวงเล็บ: ขนาดเล็กลง น้ำหนักปกติ สีเทา

## 5. องค์ประกอบ UI (UI Elements)

* การ์ดข้อมูล (Card): มุมโค้งมน (`rounded-2xl` ขึ้นไป), เงาเบา ๆ (`shadow-sm`/`shadow-md`), พื้นหลังขาวหรือครีมอ่อน
* ปุ่ม (Button): มุมโค้งมน, ปุ่มหลักพื้นทอง/เบจ ตัวอักษรดำหรือขาวตัดกันชัด, ปุ่มรองพื้นขาวขอบเทาอ่อน
* ตาราง (Table — ใช้บนคอมพิวเตอร์): เส้นบาง สีเทาอ่อน, แถวสลับสีพื้นหลังจางๆ เพื่อช่วยอ่านแนวนอน
* การ์ดสินค้า (Product Card — ใช้บนมือถือ): รูปสินค้าด้านบน, ชื่อสินค้า, ตัวเลขสำคัญแสดงเป็นแถวเด่น
* ระยะห่าง (Spacing): เว้นระยะรอบการ์ดและองค์ประกอบเยอะกว่าเว็บทั่วไป เพื่อความรู้สึก "หายใจได้"

## 6. น้ำเสียงการสื่อสาร (Tone of Voice)

* ใช้ภาษาไทยทั้งหมดบนหน้าเว็บไซต์ตามกฎของโครงการ
* คำศัพท์ภาษาอังกฤษที่จำเป็น (เช่นชื่อเทคนิคหรือไม่มีคำแปลที่กระชับ) ต้องมีคำอธิบายภาษาไทยในวงเล็บทันที ยกเว้นชื่อแบรนด์/ชื่อเทคโนโลยี
* น้ำเสียงเป็นมิตร มั่นใจ ให้ความรู้สึกเหมือนผู้ช่วยส่วนตัวด้านการคัดเลือกสินค้า ไม่ใช่ระบบรายงานบัญชีแข็งกระด้าง
