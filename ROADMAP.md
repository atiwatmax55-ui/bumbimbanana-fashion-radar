## ระยะสั้น — ใช้ข้อมูลที่มีอยู่แล้ว

### 1. คะแนน "น่าโปรโมท" (Opportunity Score)
รวมยอดขาย 7 วัน + % ค่าคอมมิชชัน + เรตติ้ง + ราคา เป็นคะแนน 0–100 ต่อสินค้า
- คำนวณใน `lib/analytics/` (ต่อยอดจาก `period-metrics.ts`)
- แสดงเป็นอันดับบนหน้าแรก + ป้ายคะแนนบนการ์ดสินค้าทุกหน้า
- เป้าหมาย: ตอบคำถาม "วันนี้ควรโปรโมทตัวไหน" ในหน้าเดียว

### 2. หน้า "มาแรงวันนี้" (Top Movers)
สินค้าที่ยอดขายเพิ่มเร็วสุดจาก snapshot ล่าสุดเทียบวันก่อนหน้า แสดงการ์ดใหญ่ 10 อันดับ พร้อม % การเติบโต

### 3. ไฮไลต์ความเปลี่ยนแปลงในหน้า "ที่บันทึกไว้"
แสดงป้ายเตือนเมื่อสินค้าที่บันทึกไว้ ยอดพุ่ง / ราคาลด / ค่าคอมเปลี่ยน เทียบกับ snapshot ก่อนหน้า

## ระยะกลาง — จุดต่างจากคู่แข่ง

### 4. กราฟประวัติราคา + ยอดขายต่อสินค้า
หน้ารายละเอียดสินค้า แสดงกราฟเส้น (Recharts) จากข้อมูล `product_sales_snapshots` ย้อนหลัง

### 5. ปุ่ม "สร้างคอนเทนต์จากสินค้านี้"
ดึงชื่อ/ราคา/จุดขายของสินค้า ไปสร้างแคปชั่น + แฮชแท็ก TikTok สำหรับช่อง "บุ๋มบิ๋มชอบกินกล้วย" (เชื่อมกับ workflow วางแผนคอนเทนต์ที่มีอยู่)

### 6. เปรียบเทียบสินค้าแบบวางคู่กัน
เลือกสินค้า 2–4 ตัว เทียบยอดขาย/ค่าคอม/แนวโน้ม ในหน้าเดียว

## ระยะยาว — จากเว็บส่วนตัว สู่ "สำหรับทุกคน"

### 7. ระบบสมาชิก (Supabase Auth)
ล็อกอิน + แยกข้อมูล "ที่บันทึกไว้" ต่อผู้ใช้ — ทำเมื่อฟีเจอร์หลักนิ่งแล้วเท่านั้น

### 8. ขยายหมวดสินค้า
เปิดหมวดความงาม/กระเป๋า/รองเท้า จาก feed เดิม (ปรับ `lib/shopee/women-fashion-filter.ts` ให้เลือกหมวดได้ ไม่ fix เฉพาะเสื้อผ้าผู้หญิง)

### 9. รวม TikTok Shop เป็นแพลตฟอร์มที่สอง
เมื่อได้อนุมัติ TikTok Shop Affiliate API (สมัครที่ partner.tiktokshop.com — ดูรายละเอียดในบทสนทนา/เอกสารเดิม):
- สร้าง `lib/data-source/tiktok-affiliate-repository.ts` ตาม Repository Pattern เดิม
- เทียบค่าคอม Shopee vs TikTok ของสินค้าใกล้เคียงกันในหน้าเดียว — ฟีเจอร์เรือธงที่ทำให้คนอื่นอยากใช้

## แผนเชื่อม TikTok Shop (อัปเดต 2026-07-08) — ทำ 3 ทางคู่ขนาน

Windsor.ai ใช้กับ TikTok Shop ไทยไม่ได้ (ยืนยันแล้ว) และต่อให้ใช้ได้ก็เห็นแค่ร้านตัวเอง
แผนจริงคือ 3 ทางด้านล่าง — **T1 เริ่มได้วันนี้** ส่วน T2/T3 ทำคู่ขนาน

### T1. หน้า "นำเข้าสินค้า TikTok" ด้วยมือ (ทำได้ทันที ฟรี ไม่ต้องรออนุมัติ)

หลักการ: เราเป็นครีเอเตอร์ TikTok Shop อยู่แล้ว → เปิด **ศูนย์ครีเอเตอร์ (Affiliate
Creator Center) → ตลาดสินค้า** จะเห็นสินค้า + % ค่าคอม + ยอดขายด้วยตาตัวเอง
→ คัดตัวที่สนใจมากรอกเข้าเว็บเรา (ไม่ใช่การ scrape — เป็นการจดข้อมูลจากบัญชีตัวเอง)

งานที่ต้องเขียนโค้ด (สั่ง AI ทำได้เลย):
- [x] หน้า `/tiktok-import`: ฟอร์มกรอกทีละตัว (ชื่อ, ราคา, % ค่าคอม, ลิงก์สินค้า,
      ลิงก์รูป, ยอดขายโดยประมาณ, หมวด) + โหมดวาง CSV หลายตัวพร้อมกัน
- [x] บันทึกเข้าตาราง `products` เดิม โดยตั้ง `source_platform='tiktok'`
      (โครงสร้างตารางรองรับอยู่แล้ว — ห้ามสร้างตารางใหม่ — ต้องรัน
      `supabase/migrations/0009_source_item_id.sql` เพิ่มก่อนใช้งานจริง ดูรายละเอียดในไฟล์)
- [x] ปรับ shopee-product-repository ให้อ่านทั้ง shopee และ tiktok
      (หรือแยก repository ตาม pattern เดิม) — แท็บ TIKTOK ในหน้าแรกจะแสดงผลเองทันที
      เพราะ `PlatformSwitch` กรองจากฟิลด์ `source` อยู่แล้ว
- [x] snapshot รายวันของสินค้า tiktok: ยอดขายอัปเดตด้วยมือผ่านหน้า import ซ้ำ
      (ระบบ snapshot เดิมรองรับ product_id ใดก็ได้ในตาราง products)

ข้อจำกัดที่ยอมรับ: ข้อมูลอัปเดตด้วยมือ ไม่อัตโนมัติ — เหมาะกับการติดตามสินค้า
ที่เลือกแล้ว 20–50 ตัว ไม่ใช่ทั้งตลาด

### T2. สมัคร TikTok Shop Affiliate API (ทางการ ฟรี — รออนุมัติ)

โครงการ TikTok Shop Affiliate Partner (TAP) **เปิดรับประเทศไทยแล้ว**
(ที่มา: partner.tiktokshop.com → Non-US TikTok Shop Affiliate Partner)

ขั้นตอน (ผู้ใช้ทำเอง — ต้องยืนยันตัวตน):
1. สมัคร Developer ที่ https://partner.tiktokshop.com (เลือก region ไทย/SEA)
2. สร้าง App ขอ scope กลุ่ม Affiliate (Creator/Partner)
3. เชื่อมบัญชี TikTok Shop Creator (ต้องมีผู้ติดตาม 1,000+)
4. ได้ App Key + App Secret → ใส่ `.env.local` (ห้าม NEXT_PUBLIC_)
5. สั่ง AI: สร้าง `lib/data-source/tiktok-affiliate-repository.ts` ตาม Repository
   Pattern เดิม — endpoint ค้นหาสินค้า open collaboration ตามหมวด/ค่าคอม/คีย์เวิร์ด

ได้อะไร: ค้นหาสินค้าที่เปิดรับ affiliate + % ค่าคอมจริง อัตโนมัติทั้งแพลตฟอร์ม
ไม่ได้อะไร: อันดับยอดขายทางการ (ต้องคำนวณเทรนด์เองจาก snapshot เหมือน Shopee)

### T3. ซื้อข้อมูลสำเร็จรูป (เร็วสุด — มีค่าใช้จ่าย ตัดสินใจทีหลังได้)

- **EchoTik** (echotik.live) — เริ่ม ~$10/เดือน มีแผนฟรีให้ลอง รองรับตลาด SEA/ไทย
- **FastMoss** (fastmoss.com) — เริ่ม ~$29/เดือน มี API รองรับไทย
- ใช้เฉพาะ **การ export/API ทางการของเขา** เท่านั้น (ห้ามเขียนโค้ด scrape)
- ข้อมูลที่ได้: อันดับสินค้าขายดี/มาแรงของไทยสำเร็จรูป → นำเข้าผ่านหน้า import

### ลำดับที่แนะนำ

1. วันนี้: สั่งทำ T1 (หน้า /tiktok-import) — TikTok tab ในเว็บใช้งานได้จริงทันที
2. สัปดาห์นี้: ยื่นสมัคร T2 (ใช้เวลาอนุมัติไม่แน่นอน — ยิ่งยื่นเร็วยิ่งดี)
3. ถ้าจริงจังขึ้น: ลองแผนฟรีของ EchoTik ก่อนตัดสินใจจ่าย (T3)
